# seis-statics v0.4.1 Release Notes

Version `0.4.1` is a patch release for the refraction datum smoothing and
multi-layer time-term ordering fixes.

## Release Contract

- public coordinate-aware floating datum smoothing API
- multi-layer sorted-position order correction
- low-level solver order contractは不変

## Public Entrypoints

The public refraction APIs remain exported from `seis_statics.refraction`:

- `resolve_smoothed_refraction_floating_datum`
- `solve_refraction_multilayer_time_terms`

The low-level solver array contract remains unchanged: package solver inputs
and outputs use sorted-position arrays prepared by the caller.

## Dependency Boundary

Runtime dependencies remain limited to NumPy, SciPy, and the Python standard
library. Application dependencies such as `app.*`, FastAPI, Pydantic,
TraceStore, artifact/job runtime code, and `seisviewer2d` are not part of the
package boundary.

## Consumer Pin

Consumers should depend on released `seis-statics` version `0.4.1`; this
release does not require a direct dependency on the `main` branch.
